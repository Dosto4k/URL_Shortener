# Описание
URL_Shortener — веб‑сервис для сокращения длинных URL‑ссылок. Позволяет создавать короткие, удобные для распространения ссылки с возможностью задания пользовательского имени.

# Основные функции
- Сокращение произвольных URL до короткой ссылки.
- Задание пользовательского имени для короткой ссылки.
- Автоматическая генерация уникальных коротких кодов.

# Шаги установки
## 1. Клонируйте репозиторий
### С помощью ***https***
```bash
git clone https://github.com/Dosto4k/URL_Shortener.git
```
### С помощью ***SSH***
```bash
git clone git@github.com:Dosto4k/URL_Shortener.git
```
## 2. Перейдите в директорию проекта
```bash
cd URL_Shortener/
```
## 3. Синхронизируйте виртуальное окружение с проектом
```bash
uv sync
```
## 4. Активируйте виртуальное окружение
```bash
source .venv/bin/activate
```
## 5. Установите pre-commit хуки (**Опционально**)
```bash
uv run pre-commit install
```
## 6. Создайте и заполните .env файл на примере .env_example
## 7. Выполните миграции
```bash
uv run manage.py makemigrations
```
```bash
uv run manage.py migrate
```
## 8. Запустите сервер для разработки
```bash
uv run manage.py runserver
```
# Использование
## API endpoints
- POST / - Создать короткую ссылку.

  Параметры:
   - url (Обязательный) - Исходный длинный URL.
   - code (Опционально) - Желаемое имя короткой ссылки.

  Пример запроса:
  ```json
  {
    "url": "https://github.com/Dosto4k/URL_Shortener/",
    "code": "short-url"
  }
  ```
  Ответ:
  ```json
  {
    "short_url": "http://127.0.0.1:8000/short-url/"
  }
  ```
- GET /{code}/ - Переход по которкой ссылке

  Пример:
  
  `http://127.0.0.1:8000/short-url/` -> перенаправляет на исходный URL `https://github.com/Dosto4k/URL_Shortener/`
